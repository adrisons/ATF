Consiste en realizar una aplicación tolerante a fallos por medio del almacenamiento periódico de la información necesaria para reiniciar una ejecución desde un punto del código dado (checkpoint). Se realizará de forma manual, se modificará una aplicación que proponga el alumno, o un programa que realice un producto de matrices de tamaño variable (si el alumno no propone ninguna otra). En caso de que el alumno desee proponer su propia aplicación, tendrá que debatir la idoneidad de esta con el profesor.
